# Linux System and Network Management Assignment

This repository contains the completion of a multi-part Linux system management assignment using Kali Linux.

## Commands Executed

* `whoami` & `uname -a`: Displayed user and system info.
* `df -h` & `free -m`: Checked disk and memory usage.
* `ps aux`, `sleep 1000 &`, `jobs`, `kill %1`: Managed background processes.
* `ping`, `ip a`, `ss -tuln`: Checked network connectivity and open ports.

All outputs were successfully saved into text files using redirection (`>`).
